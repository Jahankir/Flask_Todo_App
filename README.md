#TODO_app_using_Flask_and_MongoDB
---
To-Do-List is mini-project made with Flask and MongoDB.

## Built Using:
```bash
      Flask : Python Micro Framework
      MongoDB : Database
      PyMongo : Database Connector
      HTML5, CSS3 : For creating FrontEnd
```

## Set up environment for using this repo:
```
  Install Python(If you don't have already)

  Create a Database on MongoDB Compass
      Database Name : test, Collection Name : todo

  Install Dependencies of the application
      $ pip install -r requirements.txt
```

## Run the application
```
  Run the Flask file (run.py)
      $ python run.py

  Go to http://localhost:5000 with any of browsers and DONE !!
	    $ open http://localhost:5000

  To exit press Ctrl+C
```
